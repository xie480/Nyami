### 一、架构适配:从 CSS 到 TypeScript 主题引擎

安卓端无法直接使用 CSS 变量和 `backdrop-filter`,需要建立一个 **ThemeManager** 单例,将设计参数转化为运行时配置:

```typescript
// theme.types.ts
export interface GlassTheme {
  id: 'light' | 'dark';
  // 色彩体系
  colors: {
    pageBg: string | string[];          // 支持纯色或渐变数组
    glass: {
      bg: string;                       // rgba 格式
      border: string;
      highlightInner: string;           // inset 高光
      shimmerEdge: string;              // 顶部反光线
    };
    accent: {
      primary: string;                  // 主强调色(紫)
      secondary: string;                // 次强调色(粉/青)
      tertiary: string;                 // 第三色(蓝)
    };
    text: {
      primary: string;
      secondary: string;
      tertiary: string;
    };
    progress: {
      trackBg: string;
      fill: string[];                   // 渐变数组
      glowColor: string;                // 发光色
    };
    button: {
      secondaryBg: string;
      secondaryShadow: string;
      playBg: string | string[];
      playText: string;
    };
  };
  // 材质参数
  material: {
    blurRadius: number;                 // dp 单位
    shadowLayers: ShadowLayer[];        // 多层阴影
    backgroundOrbs: OrbConfig[];        // 光球配置
    floatingShards: ShardConfig[];      // 浮动碎片
  };
  // 动效系数
  motion: {
    hoverLift: number;                  // 悬停抬升距离(dp)
    pulseColor: string;                 // 脉冲光晕颜色
    floatDuration: [number, number];    // 浮动周期范围(秒)
  };
}

export interface ShadowLayer {
  offsetY: number;
  blur: number;
  color: string;
}

export interface OrbConfig {
  color: string;
  opacity: number;
  blurRadius: number;
  position: { x: string; y: string };  // 相对位置百分比
  animDuration: number;
  animDelay: number;
}

export interface ShardConfig {
  bg: string;
  borderColor: string;
  glowColor: string;
  rotation: number;
  animDuration: number;
  animDelay: number;
}
```

### 四、安卓端实现要点

#### 1. 玻璃模糊实现

安卓 Canvas 没有原生 `backdrop-filter`,有三种替代方案:

- **方案A(推荐)**: 使用 `RenderScript` 或 `RenderEffect`(API 31+) 对卡片下方截图做模糊,再作为卡片背景。
- **方案B**: 使用 Skia 的 `BackdropBlur` 滤镜,如果用的是 Compose 或自定义 Skia 引擎。
- **方案C(降级)**: 预计算模糊背景图,但不动态,失去“透视”感。

#### 2. 阴影绘制

用 `Paint.setShadowLayer(blurRadius, 0, offsetY, color)` 逐层绘制。

#### 3. 光球实现

径向渐变用 `RadialGradient` 着色器,配合大模糊和高透明度。

#### 4. 动效引擎

浮动动画用 `ValueAnimator` 驱动,更新 `translateX/Y` 和 `rotation`,插值器使用 `DecelerateInterpolator` 模拟 ease-in-out。

### 五、ThemeManager 示例

```typescript
// ThemeManager.ts
export class ThemeManager {
  private static instance: ThemeManager;
  private _currentTheme: GlassTheme;

  private constructor() {
    this._currentTheme = LightTheme;  // 默认
  }

  static getInstance(): ThemeManager {
    if (!ThemeManager.instance) {
      ThemeManager.instance = new ThemeManager();
    }
    return ThemeManager.instance;
  }

  get current(): GlassTheme {
    return this._currentTheme;
  }

  toggle(): void {
    this._currentTheme = this._currentTheme.id === 'light' ? DarkTheme : LightTheme;
  }

  setTheme(id: 'light' | 'dark'): void {
    this._currentTheme = id === 'dark' ? DarkTheme : LightTheme;
  }
}
```

**总结**:该方案将原本的 CSS 视觉设计完整映射为安卓端可用的 TypeScript 数据模型,所有尺寸均为比例系数或 dp 值,无固定 px 限制,您可直接在自定义 View/Skia Canvas/Compose 中消费这些配置。
